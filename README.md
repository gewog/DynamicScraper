# Ozon Parser

**Ozon Parser** — это скрипт на Python для автоматического сбора ссылок на товары с сайта [Ozon](https://www.ozon.ru/) по заданному поисковому запросу. Проект использует `undetected_chromedriver` для обхода блокировок и `selenium` для автоматизации браузера.

---

## 📌 Описание проекта

Скрипт позволяет:
- Автоматически открывать сайт Ozon.
- Вводить поисковый запрос (по умолчанию: "шторы").
- Фильтровать результаты по категории "Новинки".
- Прокручивать страницу для загрузки дополнительных товаров.
- Собирать ссылки на товары и выводить их в консоль.

---

## 🛠 Технологии

- **Python 3.10+**
- **Poetry** — для управления зависимостями.
- **Selenium** — для автоматизации браузера.
- **undetected_chromedriver** — для обхода блокировок.
- **fake_useragent** — для генерации случайных User-Agent.
